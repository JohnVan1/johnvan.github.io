# johnvan.github.io

This repo powers your GitHub Pages site.

## Quick setup

1. Replace `assets/profile.jpg` with your real profile picture.
2. Replace `assets/resume.pdf` with your real resume PDF.
3. Edit `index.html` and update:
   - your name
   - the short description
   - the LinkedIn and GitHub URLs
4. Add projects by copying a card in the **Projects** section of `index.html`.

## Publish on GitHub Pages

- Repo name must be `johnvan.github.io` (exactly).
- In GitHub, go to **Settings → Pages**.
- Set **Build and deployment** to “Deploy from a branch”.
- Select branch **main** and folder **/(root)**.
